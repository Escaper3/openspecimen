caTissue v1.1.p1 is brought to you by Persistent Systems, a NCI certified caBIG Support Service provider. This version of caTissue is free to use and is an open source application. The new features in this release are either funded by Persistent or by our various customers. The major features in the caTissue Suite v1.1.p1 release are:
  • Search functionality like Google
  • Auto generation capability for PPI field
  • Performance enhancements
  • Support for UK locale
  • Provision for #54 bug fixes

  • Support for another open source de-id tool
  • Continued improvement for better UI resolution support
  • Profiling for memory and performance enhancements
  • Other critical bug fixes and enhancements identified by our customers
